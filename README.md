## 📋 주요 기능

### ① 반보존적 복제 (4단계)
- 원래 DNA의 이중나선 구조
- 수소결합이 끊어지며 가닥 풀림
- 주형 가닥을 따라 새 가닥 합성
- 원래 가닥 + 새 가닥으로 된 동일한 DNA 2개 완성
- **색깔 구분**: 파랑(원래 가닥) / 분홍(새 가닥)

### ② 복제 분기점 상세 (5단계)
- 복제 분기점의 Y자 형태
- RNA 프라이머 결합
- **선도 가닥**: DNA 중합효소가 연속적으로 합성 (5'→3')
- **지연 가닥**: 오카자키 절편으로 끊어서 합성
- DNA 연결효소가 조각 이어붙이기
- **효소 표시**: 중합효소(주황), 프라이머(초록), 연결효소(초록)

### ③ 염기 짝짓기 실습
- 학생이 직접 **DNA 중합효소 역할** 수행
- 주형 가닥의 염기에 맞는 상보 뉴클레오타이드 선택
- A=T, G≡C 상보결합 규칙 학습
- 뉴클레오타이드 구조(당·인산·염기) 시각화
